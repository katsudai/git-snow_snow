## サイト概要
### サイトテーマ
<スノーボードが好きな人がスノーボードで滑っている画像など投稿したり、スノーボードに関する情報を共有できるサイト>

### テーマを選んだ理由
<スノーボードが好きな人が集まれる場所の提供。スノーボードをしたいけど一緒に行く人がいない、など
一人でスノーボードをすることが多い人でも情報をいろいろな人と共有できるサービスがあれば便利と考えたため>

### ターゲットユーザ
